## 一键安装Trojan-Go脚本


**输入**

**1 证书文件名**

**2 Trojan-Go的版本号**

**3 伪装的网站的文件**

**4 Trojan-Go的密码及域名**


**自动安装Nginx，并将侦听端口号改为8443。**

**新建 trojango 用户, 使用此用户启动 trojango,trojan-Go的默认开启CDN,并侦听443端口. **
